# Imágenes de Planes y Precios

Este directorio contiene las imágenes para los planes de precios.

## Archivos necesarios:

- `box-type-m.png` - Imagen para el tamaño M
- `box-type-l.png` - Imagen para el tamaño L  
- `box-type-xl.png` - Imagen para el tamaño XL

## Cómo agregar las imágenes:

1. Coloca tus imágenes en este directorio con los nombres exactos mencionados arriba
2. Las imágenes se mostrarán automáticamente en la sección "Tamaños de consignas"
3. Tamaño sugerido: relación 4:3, <500 KB (usa WebP si puedes)

## Editar textos y descripciones:

Para cambiar títulos o descripciones, edita `src/assets/i18n/es.json` (sección `lockerSizes`).

Para instrucciones detalladas, ve a [docs/how-to/imagenes-planes.md](docs/how-to/imagenes-planes.md).
